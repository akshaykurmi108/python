# Python for Beginners (with OOP)

A collection of simple Python programs to help beginners learn Python basics along with Object-Oriented Programming concepts.

## What's Inside
**Basics**
- Variables, data types, and operators
- Conditional statements (if-else)
- Loops (for, while)
- Functions and basic I/O
- Basic data structures (Lists, Tuples, Dictionaries)

**OOP (Object-Oriented Programming)**
- Classes and Objects
- Constructors (`__init__`)
- Inheritance
- Polymorphism
- Encapsulation

## Requirements
- Python 3.x installed

## How to Run
1. Clone the repository
2. Run any script using

## Purpose
This repo is meant for anyone starting out with Python and wants hands-on, simple examples to practice both basics and OOP.
